Real-Time Stock Market Data Pipeline

Project Overview

This project demonstrates an end-to-end real-time data engineering pipeline that streams stock market data using Apache Kafka, processes it with Spark Streaming, stores it in PostgreSQL, and visualizes insights through Power BI dashboards.

Tech Stack

 Python
 Apache Kafka
 Apache Spark Streaming
 PostgreSQL
 Power BI
 Docker
